Sobre

Os dados recolhidos são provenientes da produção de uma fábrica ao longo de uma semana.

O desafio propõe que o participante faça uma análise exploratória dos dados e gere gráficos para visualização da produção diária de uma das máquinas. Além do que foi pedido, nesta aplicação há um gráfico extra que calcula a média do tempo gasto entre produções consecutivas todos os dias utilizando o campo sis (speed in seconds), a fim de verificar se houve perda de produtividade devido à lentidão da máquina e, também, se esta média tem tendência de crescimento ao longo da semana.
